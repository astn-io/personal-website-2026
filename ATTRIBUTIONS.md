## Technology

### [Astro](https://astro.build/)

Static site framework handling file-based routing, content collections, and server-side rendering. Astro components (`.astro`) are used for layouts, pages, and structural elements.

### [Svelte](https://svelte.dev/)

Component framework used for interactive UI elements. Uses Svelte 5 runes (`$state`, `$props`) for reactivity.

### [Sass](https://sass-lang.com/)

CSS preprocessor used for all styling. No CSS framework — styles are fully custom using SCSS with CSS custom properties.
